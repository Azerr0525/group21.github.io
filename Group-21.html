Define the following
ram (random access memory)
-Random Access Memory

rom (Read-only memory
-is a type of computer storage containing non-volatile,
permanent data that, normally, can only be read, not written to.

Ai (Artificial intelligence)
-is the simulation of human intelligence processes by machines,
especially computer systems.

CD (Compact Disc)
-a small plastic disc on which data or music is stored

USB (Universal Serial Bus)
-technology used to connect computers with peripheral devices.

CPU (Central Processing Unit)
-The component of a computer system that controls
the interpretation and execution of instructions.

FAQ (Frequently Asked Question)
-used to refer to a list of answers to typical questions that
users of a Web site might ask

GB (Gigabyte)
-A gigabyte is a specific unit of data that's equal to about
1 billion bytes of data.
